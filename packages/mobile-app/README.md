# MoDos Mobile App

## Install, Develop, Build, Deploy
### Pre Steps
- Install Angular CLI
- Install Android Studio
- Clone this repository

### Install
```shell
git clone 'modos repo...'
cd packages/mobile-app/
npm ci
```

### Develop
```shell
# on web
npm run start

# on android
npm run dev:android:watch
```

### Build
```shell
npm run build

# for android
npx cap copy && npx cap update && npx cap open android
```
